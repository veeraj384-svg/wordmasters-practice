About / Instructions

How to use this site and how to add new practice pages.

How students should use it

  1. Start at Home.
  2. Do Dropdown Story first (fast review).
  3. Then do Week 1 Quiz (definitions + context).
  4. Finish with Week 2 Challenge (write sentences).

How parents/teachers can use it

  • Have the student explain why the chosen word fits.
  • For Week 2, use the rubric at the bottom of the page (#teacher section).
  • Repeat until the student can use each word naturally in conversation.

Add more files (the easy way)

In GitHub: Add file → Create new file, name it like week3.html, paste content, and commit. Then add a new button in index.html linking to week3.html.

Troubleshooting

  • If a button “does nothing,” it usually means the file doesn’t exist or the filename doesn’t match.
  • Cloudflare Pages redeploys automatically after you commit changes.
  • If you want missing pages to show a clear message, add a 404.html file.
